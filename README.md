## 🧠 Features

- 🕒 Analog Clock with smooth animation  
- 🧮 Digital Clock with AM/PM format  
- 🌗 Toggle button for **Light / Dark** themes  
- 💾 Saves selected theme using `localStorage`  
- 📱 Fully responsive design  
- 🎨 CSS variables for easy customization
